# 路基厚度

## 规范来源
JTG F80/1-2017 第4.2.x条

## 适用范围
适用于路基结构层厚度检测场景，用于对比实测厚度与设计厚度，支持现场快速判定与数字化归档。

## 检测步骤
1. 现场采集实测厚度。
2. 录入设计厚度。
3. 系统将实测厚度写入输出字段并执行判定规则。
4. 完成试验与监理签名，形成节点级与容器级证明。

## 合格标准
厚度必须满足：`measuredThickness >= designThickness`。

## 输入参数
- `measuredThickness`：实测厚度（mm）
- `designThickness`：设计厚度（mm）

## 系统对接说明
- 规则中的 `value` 支持直接引用输入字段名（例如 `designThickness`）。
- 导入后可直接在现有表单、规则、proof 流程中执行。
- 无需修改页面结构即可参与多 SPU 编排。
