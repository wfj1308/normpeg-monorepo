# 路基压实度（土质）

## 规范来源
JTG F80/1-2017 第4.2.1条

## 适用范围
适用于公路工程路基土质填筑层压实度检测，适用于现场灌砂法检测结果的系统化执行、判定与归档。

## 检测步骤
1. 记录试坑灌砂质量与体积。
2. 录入含水率与最大干密度。
3. 系统执行湿密度、干密度与压实度计算。
4. 系统按规范阈值进行自动判定。
5. 完成试验与监理双角色签名后归档。

## 合格标准
压实度必须满足：`compactionDegree >= 93%`。

## 输入参数
- `massHoleSand`：灌入试坑砂质量（g）
- `volumeSand`：试坑体积（cm3）
- `moistureContent`：含水率（%）
- `maxDryDensity`：最大干密度（g/cm3）

## 系统对接说明
- 本规范项可作为标准 SPU 导入系统并自动注册。
- 导入后复用统一 Runtime 执行引擎，不需要页面特例开发。
- 结果可进入 Container 工作流，支持 attempt 历史、Gate 判定、签名与 Proof 存证。
